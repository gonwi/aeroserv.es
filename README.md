# Aeroserv website

Stack:
- Next.js 16
- Tailwind CSS v4
- shadcn/ui-compatible structure
- App Router
- ES / EN language switcher dropdown

## Run

```bash
npm install
npm run dev
```

## Notes
- Main routes: `/es`, `/en`, `/es/servicios`, `/es/sobre-nosotros`, `/es/contacto`
- Real logo included in `public/logo-aeroserv.png`
- Legal pages are placeholders pending final approved text
